# Портфолио Data Analyst

## Обо мне
Data Analyst с фокусом на BI-дашборды, визуализацию данных и аналитические исследования.  
Имею опыт работы с финансовыми данными и бухгалтерской отчетностью.  
Специализируюсь на преобразовании данных в инструменты для принятия решений и построении аналитики с нуля.

---

## Проекты
Ниже представлены проекты, демонстрирующие аналитическое мышление, работу с данными и построение BI-решений.

---
 

### 1. Эволюция профессии аналитика (Dataviz)
Анализ развития ролей аналитиков и их разделения на специализации (BI, Data, ML и др.).  
Модель представлена в виде структуры роста (branching model).  
→ [Открыть проект](analytics_role_evolution_dataviz)

---

### 2. LEGO Dashboard (Power BI)
Аналитическое исследование LEGO-наборов с целью оценки совместимости деталей между различными франшизами.

В проекте:
- рассчитана степень пересечения деталей (индекс Жаккара)
- построен рейтинг «гибкости» франшиз
- выявлены ключевые узлы (например, Batman как универсальная тема)

→ [Открыть проект](lego_parts_reuse_analytics_jaccard_similarity)

---

### 3. SWOT-анализ маркетинговых кампаний
Проект, направленный на разработку аналитической методики оценки маркетинговых стратегий.

Фокус:
- агрегация данных
- сравнительный анализ
- выявление ключевых факторов эффективности

→ [Открыть проект](power_bi_pet_projects)

---

### 4. Логистика склада (Power BI)
Анализ складских процессов с выявлением узких мест и неэффективностей.

🏆 Приз зрительских симпатий за лучшую визуализацию (Мария Гришина)

→ [Открыть проект](power_bi_pet_projects)

---

 

### 5. Arctic Dashboard — Northern Development Index
Интерактивный дашборд для оценки развития 128 арктических населённых пунктов России на основе индекса разработанного для конкурса индекса NDI.
 

### Структура индекса:
- POAD — 35%
- Market Access — 20%
- Consumption — 15%
- Accessibility — 15%
- Climate — 10%
- Mobility — 5%

→ [Открыть репозиторий](https://github.com/DaryaPodakova/sberindex_dashboard)
