# LEGO DataViz: Анализ совместимости деталей между франшизами

## Описание проекта

Проект представляет собой аналитическое исследование LEGO-наборов, направленное на оценку **совместимости деталей между различными франшизами**.

На основе очищенных данных по составу наборов была рассчитана степень пересечения деталей между темами (например, Batman, Star Wars, Ninjago и др.) с использованием **индекса Жаккара**.

Результат визуализирован в виде:
- рейтинга «гибкости» франшиз (насколько легко их комбинировать)
- графа пересечений между темами
- выявления центральных «узлов» (например, Batman как наиболее универсальная тема)

## Методика

1. **Подготовка данных**
   - удалены технические категории деталей (шум)
   - исключены уникальные элементы (не влияющие на переиспользование)
   - отобраны релевантные франшизы (поп-культура / geek-сегмент)

2. **Формирование датасета**
   - каждая франшиза представлена как множество деталей

3. **Расчет метрики**
   - рассчитан **Jaccard Index** для каждой пары франшиз:
     - показывает долю общих деталей
     - отражает реальную «совместимость» наборов

4. **Агрегация**
   - рассчитан общий индекс «гибкости» франшизы:
     - насколько она пересекается с другими

## Бизнес-задача (интерпретация)

### 1. Анализ переиспользования компонентов
Оценить, насколько элементы (детали) могут использоваться повторно между продуктами.

→ аналог:
- унификация компонентов в производстве
- reuse в продуктовой линейке

---

### 2. Выявление «универсальных» продуктовых линий
Определить франшизы, которые:
- легче комбинируются
- лучше подходят для кросс-продаж

→ пример:
- Batman = high compatibility hub

---

### 3. Оптимизация продуктового портфеля
Разделение франшиз на:
- **гибкие (flexible)** — можно расширять и комбинировать
- **замкнутые (isolated)** — требуют отдельного развития

→ аналог в бизнесе:
- модульные vs монолитные продукты

---

### 4. Поддержка решений по коллаборациям
Понять:
- какие франшизы лучше подходят для кроссоверов
- где синергия максимальна

→ аналог:
- партнерства
- совместные продукты

---

### 5. Снижение затрат на разработку
Высокая совместимость деталей → потенциально:
- ниже себестоимость
- быстрее разработка новых наборов

---

## Краткое резюме

Аналитическая модель оценивает, насколько элементы продуктов (детали LEGO) могут переиспользоваться между различными линейками, и выявляет ключевые «узлы совместимости», что позволяет принимать решения по развитию портфеля, коллаборациям и оптимизации затрат.
